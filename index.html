About

Engineer + Creative

I build systems where engineering rigor meets creative craft. My work spans frontend interfaces with cinematic animations, AI agent orchestration, and production infrastructure that scales.

On the frontend, I architect performant interfaces using React, TypeScript, and GSAP. I've built award-winning sites with 60fps animations and real-time dashboards handling 50,000+ events per second. Performance isn't just metrics — it's about perceived speed and intentional motion.

On the systems side, I design AI agent workflows using LangChain and build MCP servers that expose custom tools to LLMs. I think in terms of orchestration patterns, state management, and graceful degradation. The hard problems are always about coordination, not just computation.

I also experiment with Blender for 3D work, generative systems, and visual explorations. The best work happens at the intersection of disciplines — where code feels like craft and systems feel like art.

How I Think

Approach

Philosophy 01

Systems Thinking

Code should feel inevitable. I design systems where components compose naturally, state flows predictably, and errors are handled gracefully. The best architectures disappear — they just work.

Philosophy 02

Performance vs. Experience

Real performance is perceived speed. A 200ms operation that feels instant beats a 100ms operation that feels janky. I optimize for human perception, not just Lighthouse scores.

Philosophy 03

Engineering + Design

My best work happens at the intersection. Understanding visual hierarchy makes me write better components. Understanding technical constraints makes better design decisions.
